# Cheat Sheet

## Vim

- `gV` : select latest pasted taxt

- `sb` : fzf buffers
- `sf` : fzf files
- `sh` : fzf history
- `st` : fzf tags
- `sl` : fzf lines
- `sc` : fzf commits
- `sa` : fzf ag

- `sd` : Dash search


- `:Today` : print today
